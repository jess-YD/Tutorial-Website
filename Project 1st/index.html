CONDITIONALS

var RS = 10;
                    if (RS % 2 ===0 && RS % 5 ===0) {
                        console.log('rocket science');
                    } else if (RS % 5 === 0) {
                        console.log('science');
                    } else if (RS % 2 === 0) {
                        console.log('rocket';
                }

Conditionals are important because they determine if a code can be run or not. One of the most important types of conditionals are the ones that use the if/ else statements that are enclosed in (). The example above demonstrates that if RS is true the else if statements will be runned. You can continue an if statement with an else statement to add another conditional.

FOR LOOPS


                        for(i = 10; i>0;i-=1){
                            console.log(i);
                        }

If you want to run a code repeatedly until the condition is met then a loop will run. A for loop involves counting, but can get you to the point faster with only 3 lines of code. The example shows that counting 10 to 0 with a for loop will run faster instead of having to literally write all of the numbers out.

FUNCTIONS


                        let i = 1;
                        function counting() {
                       while(i<= 10) {
                           console.log(i),
                       i ++;
                       }
                   }
                   counting();

Functions do not mean anything if they are not called. Like the example above, the example will run numbers 1 throguh 10, but in order for that to happen it will be called by let i = 1;. Then, followed by function counting() the function while(i<=10) will be console.log and will do a for loop of i ++; and lastly in order for it to execute, it has to be called by counting();
